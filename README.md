# Indian Stock Analysis Tool

An open-source Python tool for analyzing Indian stock market data and generating basic trading signals.

## Features
- Fetch stock data using yfinance
- Calculate Simple Moving Averages (SMA 5 & 10)
- Generate basic BUY/SELL signals

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python app.py
```

Example:
```
RELIANCE.NS
```

## Tech Stack
- Python
- Pandas
- yfinance
